# Data Science Club Website

This project is a Next.js + Tailwind website for the Data Science Club.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm start
```

## Pages

- `/` — Home with hero, notices, and upcoming events
- `/about` — Mission, vision, and core values
- `/initiatives` — Club programs and initiative detail pages
- `/team` — Team members and contact info

## Notes

- Use `pages/index.tsx` to edit latest notices and upcoming events.
- Use `data/initiatives.ts` to add or update initiative cards.
- Put images and logos in `public/` and reference them by path.
- The app is ready for deployment to Vercel or GitHub Pages after a normal Git push.
